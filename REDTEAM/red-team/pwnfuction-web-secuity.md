# Pwnfuction Web Secuity

* Trickest is a tool that allows Red Team activity and ethical hackers to create automated tasks.
* To play with Trickest, log in with Google using Community Edition. It has RU (Run Units), which are points that will be reduced when the nodes are being used.
* RU is a prepared resource unit used in Trickest to execute workflows.

## Creating a Workflow

* Find the space or project where you want to create a workflow.
* Click the **Create** button.
* Set the name and optional description.

When the workflow is created, click on the workflow card to open the Workflow page. Then click the **Open Workflow** button to start building its content in Workflow Builder, following the steps shown in "Building a Workflow."

## Fleet Management

* By default, Trickest provides a machine with 2 CPU and 4 RAM, where the nodes can be run, or it can use the local machine / third-party resources using Fleet Management.
* For third-party machines, Trickest uses port 3000 to communicate.
* Click on **ADD Machine**.
* Copy and paste the code in the local machine.
* Download the `init.sh` using `curl`.
* Export the key into PATH$.
* Give execute permission and execute the `init.sh`.
