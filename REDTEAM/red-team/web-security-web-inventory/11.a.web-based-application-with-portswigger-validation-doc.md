| ## **WebGoat** |
| -------------- |

WebGoat is a deliberately insecure web application that allows developers to test vulnerabilities in Java-based applications. The application is maintained by OWASP and is designed to teach web application security lessons. The exercises are intended to help people learn about application security and penetration testing techniques.

WebGoat was created by Bruce Mayhew while he was working at Aspect Security. He created it as a way to teach developers about common web application security flaws and how to prevent them. Today, WebGoat is maintained by the Open Web Application Security Project (OWASP) and is a popular tool for learning about web application security.

**Some of the injection vulnerabilities in WebGoat include:**

* Command Injection (cmdi)
* SQL Injection (sqli)
* LDAP Injection (Idapi)
* XPATH Injection (xpathi)
* Cross Site Scripting (xss)
* Path Traversal (pathtraver)
